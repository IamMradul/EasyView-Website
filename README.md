## рџљЂ Core Features

- **Jargon Decoder**: Converts dense/technical text into plain language.
- **Dyslexia Reading Mode**: Better spacing, overlays, and readability-first presentation.
- **Sensory Shield**: Reduces motion and visual overload for calmer browsing.
- **Text-to-Speech Support**: Listening-first experience for easier content consumption.
- **Interactive Demo**: Before/after transformation of complex text.
- **Reduced Motion Toggle**: Respects and allows control of motion preferences.

## рџ§  Accessibility-First Highlights

- Motion-aware UI with `prefers-reduced-motion` handling.
- Clear visual hierarchy and high readability content sections.
- Keyboard-friendly interactive controls.
- Dedicated accessibility guide route for users and contributors.
